Information collected automatically:
• Usage data: Which features you use, how long you run auctions, and general interaction data to help us improve the platform.
• Device & browser info: Browser type, operating system, and IP address for security and analytics purposes.
• TikTok LIVE data: When connected via TikTok's API, we receive real-time gifting event data (username, coin amount, timestamp) to power the leaderboard. This data is processed in real-time and is not permanently stored beyond your current session.

We use the information we collect to:

We do not sell your personal data to third parties. We do not use your data for advertising purposes.
